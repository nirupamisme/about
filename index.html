Lid driven cavity


This is a standard benchmark problem. Lid of the square cavity is moving with a velocity, u = 1 units/s, v = 0 units/s and all the other walls are stationary. The Reynolds No. for the flow is given as, Re = 100. Streamfunction and Vorticity method was used for solving the problem.

Da-Laval Nozzle


A standard subsonic-supersonic isentropic nozzle is solved using the conservation form of governing equations. Maccormack's technique was used (Explicit Scheme) for the solution. Numerical diffusion was used for getting a stable solution with shock capturing.

Link to the source code

Lorenz Attractor


The Lorenz attractor is a set of chaotic solutions to the Lorenz system, which is a system of ordinary differential equations originally developed by Edward Lorenz in 1963. The equations model the atmospheric convection and exhibit chaotic behavior for certain parameter values.

Link to the source code

Turbulent Channel Flow


The fully-developed turbulent channel flow is solved numerically using k−ε, k−ω and RNG k−ε turbulence model and the comparison is plotted. The mean flow is one-dimensional (1D).
